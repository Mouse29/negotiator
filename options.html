Referer

For a request from http://example.com/name/of/page.html to http://target.com/subpage.php

Remove - Remove the "Referer:" header completely, naturally occurs when you enter the address manually.

Clean - Only send the domain of the referrer, http://example.com

Clean Destination - Send the domain of the destination, http://target.com

Destination - Send the entire destination, http://target.com/subpage.php

Cookie

Remove - Prevent cookies from being set and, if already set, being sent to the remote server.

*/* - Send the "*/*" Accept header rather than the browser specific one.
