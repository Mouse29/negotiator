 Request Filter

Filters

Format:

header: value

header is the http header. value is either the new value of the header or one of the following keywords:

pass
Send the header unmodified, for User-Agent that is
remove
Remove the header.
clean
Only send the domain of the referrer, http://example.com
destclean
Send the domain of the destination, http://target.com
dest
Send the entire destination, http://target.com/subpage.php
random
Random User-Agent for every request:
